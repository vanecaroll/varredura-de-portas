🖥️ Verificação de Portas com PHP

Este script em PHP simula a varredura de portas de rede (1 a 20) utilizando o loop while.

🔁 Funcionamento:

Percorre as portas de 1 até 20
Exibe na tela: A verificar porta X

🚫 Regras aplicadas:

🔄 Portas múltiplas de 5 (5, 10, 15...) são ignoradas com continue
🛑 Ao atingir a porta 18, o processo é interrompido com break

💡 Conceitos utilizados:

while → repetição
if → decisão
continue → pula para a próxima iteração
break → encerra o loop
